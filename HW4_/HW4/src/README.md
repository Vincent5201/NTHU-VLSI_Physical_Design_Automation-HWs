## How to compile
In "HW4/src", enter the following command:
* `$ make`: An executeble file "hw4" will be generated in "HW4/bin".
* `$ make clean` Remove files in "HW4/bin".

## How to run
In "HW4/bin", enter the following command:
`$ ./hw4 <input file> <output file>`
E.g., in "HW4/bin", enter:
`./hw4 ../testcase/public1.txt ../output/public1.out`
