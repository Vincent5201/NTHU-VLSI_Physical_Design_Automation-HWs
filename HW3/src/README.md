## How to compile
In "HW3/src", enter the following command:
* `$ make`: An executeble file "hw3" will be generated in "HW3/bin".
* `$ make clean` Remove files in "HW3/bin".

## How to run
In "HW3/bin", enter the following command:
`$ ./hw3 <input LEF> <input DEF> <output DEF>`
E.g., in "HW3/bin", enter:
`./hw3 ../testcase/public1.lef ../testcase/public1.def ../output/public1.def`
