## How to compile
In "HW2/src", enter the following command:
* `$ make`: An executeble file "hw2" will be generated in "HW2/bin".
* `$ make clean` Remove files in "HW2/bin".

## How to run
In "HW2/bin", enter the following command:
`$ ./hw2 <input file> <output file> <number of partitions>`
E.g., in "HW2/bin", enter:
`./hw2 ../testcase/public1.txt ../output/public1.2way.out 2`
